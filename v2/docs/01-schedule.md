## 🌅 ช่วงเช้า — รายละเอียด

### 09:00–09:20 | เปิดงาน + โชว์ปลายทาง
โชว์ของจริง 1 ตัวที่จบ flow แล้ว (model รันบนบอร์ด ตอบ output ได้) ให้เด็กเห็นว่าจบวันนี้จะได้อะไร แล้วบอกว่าวันนี้ "ทุกทีมเดินเส้นเดียวกัน ไม่มีใครต้องเลือกของยากตั้งแต่เช้า"

### 09:20–09:40 | สองสมองของ UNO Q
ปูให้เข้าใจก่อนต่ออะไรทั้งสิ้น เพราะมันอธิบายว่าทำไม input แต่ละแบบต่อคนละทาง:
- **MCU side (real-time):** อ่าน Modulino / sensor / คุม output ทันที
- **Linux side (compute):** รัน AI, กล้อง USB, ไมค์ USB, คุยกับ Edge Impulse
- สองฝั่งคุยกันผ่าน bridge — จำคอนเซปต์นี้ไว้ บ่ายจะได้ไม่งงว่า data ของ sensor ไปโผล่ Edge Impulse ได้ยังไง

### 09:40–10:30 | Reset + First Boot ★
บอร์ดวนมาจากที่อื่น รหัส/ชื่อไม่รู้ → ทุกทีม reset เป็นของตัวเอง
ทำตาม [`labs/morning-uno-q.md`](../labs/morning-uno-q.md) ส่วน **Reset**
จบช่วงนี้ทุกทีมต้องมี: บอร์ดที่ตั้งรหัสเอง + ตั้งชื่อ `team-XX-q` + ต่อ Wi-Fi ได้

### 10:30–11:00 | Input 1 — Modulino sensor
ต่อ Qwiic + อ่านค่าให้ขึ้นจอผ่าน App Lab จบที่ "เห็นค่าขยับตามที่ขยับ sensor"

### 11:00–11:30 | Input 2 — USB Webcam
เสียบผ่าน **powered USB hub** + เห็นภาพสด ⚠️ ของเดิมพังบ่อยตรง power ไม่พอ — ย้ำ hub มีไฟเลี้ยง

### 11:30–11:50 | Input 3 — USB Mic
เสียบ + จับเสียงได้ จบที่ "พูดแล้วเห็น level ขยับ"

### 11:50–12:00 | เชื่อมเช้า→บ่าย
สรุปสั้น: "3 input ที่เพิ่งต่อ บ่ายเราจะเลือก 1 อย่างส่งเข้า Edge Impulse ไปสอน AI" — กันเช้ากับบ่ายขาดจากกัน

## ☀️ ช่วงบ่าย — รายละเอียด

### 13:00–13:30 | Edge Impulse 101 + bias
pipeline 4 ขั้น (Collect → Train → Deploy → Test) + bias สั้น ๆ ("เก็บจากคนเดียว/ฉากเดียว AI จะจำผิดจุด")

### 13:30–14:00 | เชื่อม UNO Q เข้า Edge Impulse ★
ทำตาม [`labs/afternoon-edge-impulse.md`](../labs/afternoon-edge-impulse.md) ส่วน **Connect**
จบช่วงนี้ทุกทีมต้องเห็น input ตัวเอง (sensor/ภาพ/เสียง) ไหลเข้า Studio จริง ถ้ายังไม่เห็น **อย่าเพิ่งไปเก็บ data**

### 14:00–14:45 | เก็บ data + train V1
เลือก 1 input → เก็บตามเป้า → create impulse → train V1

### 14:45–15:30 | Deploy + inference จริง ★
build จาก Edge Impulse → ลง UNO Q ผ่าน App Lab → รันบนบอร์ดจริง จบที่ "ป้อน input จริงแล้ว model ตอบได้"

### 15:30–16:00 | เทส 10 cases
จดลง [`team-template/afternoon/predictions.csv`](../team-template/afternoon/predictions.csv) — 5 เคสปกติ + 5 เคสท้าทาย

### 16:00–16:30 | โชว์สั้น + ส่งงาน
โชว์ทีมละ 1–2 นาที + ส่งงานผ่าน fork (ดู [`team-template/README.md`](../team-template/README.md)) + เกริ่น Day 2 ปล่อยให้ทำเอง
